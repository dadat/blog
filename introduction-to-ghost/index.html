Introduction to Ghost

Thursday, December 03, 2015

An idea popped up when I tried out GitHub Pages.
"Why don't I host my blog on it?", I said. First one I tried out was Jekyll - written in Ruby, this thing had a ton of documentation ready for deployment. With one major problem:

It is not native to Windows(which I am unfortunate enough to be using).

Next up on my list was Hexo. It had plenty of themes ready, and the community was alive, but I could not place my finger on what was bothering me about it. Installation was quick, although it seemed too easy. Easy is fine, but when you're on Windows, you will know that easy comes at a price.

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server

A few minutes after installation, the problems had begun.
